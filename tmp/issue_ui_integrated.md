## 🎯 目的 / 背景
WBS と連動したリアルタイム・タイムラインを実現。プロジェクトの進捗を視覚的に一瞬で把握可能にする。

---

## 🛠 実装設計 (Implementation Design)

### 1. ガントチャート・コンポーネント (`GanttChart.tsx`)
- **Library**: `recharts` または `d3` ベースのカスタム SVG。
- **Props**: `project_id`, `items[]` (階層化されたタスクリスト)。
- **Logic**: タスクの `start_date`, `due_date` を X 軸（時間軸）にプロット。

### 2. 進捗計算ロジック (`ProgressHelper.ts`)
- **Input**: 子タスクリスト。
- **Calculation**: `SUM(actual_hours) / SUM(estimated_hours) * 100`。
- **Trigger**: 子タスクのステータス変更時に、親タスクの進捗率を自動再計算。
