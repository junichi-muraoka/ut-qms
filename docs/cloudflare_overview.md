# インフラ知識：Cloudflare とは？ (Cloudflare Overview)

## 1. 概要
本プロジェクト（UT-QMS）をウェブ上に公開するために採用している「Cloudflare（クラウドフレア）」について、初心者向けに解説します。

## 2. Cloudflare の 3 つの大きな役割

### ① アプリを表示する (ホスティング)
自分の PC で作ったアプリを、世界中の人が 24 時間見られるように「インターネットの広場（サーバー）」に置いてくれます。

### ② アプリを守る (セキュリティ)
インターネット上の攻撃や不審なアクセスを、マンションの警備員のように自動で追い払ってくれます。

### ③ 表示を速くする (高速化)
世界中にある拠点を使って、アクセスしてくる人に一番近い場所からアプリを表示するため、一瞬でページが開きます。

## 3. なぜ「無料」なのか？
Cloudflare には非常に強力な**無料プラン (Free Plan)** があります。

- **仕組みによる防御**: 無料枠（1 日 10 万リクエストなど）を超えると「自動で課金」されるのではなく、**「その日の動作がエラーで止まる」** 仕組みです。
- **クレジットカード不要**: 支払い情報を登録せずに使い始めることができるため、意図しない課金の心配がありません。

## 4. 採用の理由
今回使用しているバックエンド技術（Hono）と非常に相性が良く、**「世界最速クラスの性能を、完全無料で手に入れる」** ことができるため採用しました。
